### 선택 과제
- margin
- padding
둘다 여백의 개념이다. padding은 **안쪽 여백**, margin은 **바깥쪽 여백**이다. 예를 들어 버튼을 생각해보면 ``` button { padding : 20px; } ``` 코드는 버튼의 안쪽 여백이 커진다. 게임에서 어떤 것을 클릭한다고 했을 때 그 버튼을 크게 해준다고 이해했다. 다음은 margin인데 margin은 바깥쪽 여백을 크게 해주는 기능인데 버튼의 크기가 커지는 것이 아닌, 버튼 주변에 빈 공간을 만들어준다. 코드의 형식은 ``` button { margin : 20px; } ```와 같다.
